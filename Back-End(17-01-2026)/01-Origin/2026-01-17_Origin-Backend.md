## What are packages

Packages are the publicly available code which is used by fellow developers to add a specific funtionality to there project without re-writting the whole code

all the npm packages are available in *NPMJS.COM* 

## How to install packages 
`npm i "package name" `

## How to use packages
`require that package in your server.js file then call to use it`
